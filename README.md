# Test Python Repository

This is a simple test Python repository demonstrating a basic "Hello World" application with unit tests.

## Project Structure

- `src/`: Contains the source code
- `tests/`: Contains unit tests
- `requirements.txt`: Lists project dependencies (currently empty as no external dependencies are required)

## Setup

1. Clone the repository:
   ```
   git clone https://github.com/af-bluec/test-git-bot-pub.git
   cd test-git-bot-pub
   ```

2. (Optional) Create and activate a virtual environment:
   ```
   python -m venv venv
   source venv/bin/activate  # On Windows, use `venv\Scripts\activate`
   ```

3. Install dependencies (if any):
   ```
   pip install -r requirements.txt
   ```

## Running the Application

To run the "Hello World" application:

```
python src/hello_world.py
```

## Running Tests

To run the unit tests:

```
python -m unittest discover tests
```
